# 肖四第一套卷 34题(马原)

## 题目

**34. 结合材料回答问题：**

### 材料1

习近平总书记强调："要把满足人民对美好生活的向往作为科技创新的落脚点，把惠民、利民、富民、改善民生作为科技创新的重要方向。"

向善惠民，始终是科技创新的价值取向。

在乡村沃野，智能高效稻麦联合收割机1分钟收割600公斤作物，大幅提高夏收秋收效率；在医院手术室，国产骨科手术机器人、腔镜手术机器人投入应用；在养老机构，智能腕表和智能拐杖等装备优化养老服务……科技创新与百姓生活的融合，更好满足人民群众对美好生活的向往。

恩格斯曾说，"社会一旦有技术上的需要，则这种需要就会比十所大学更能把科学推向前进。"我国有14亿多人口，中等收入群体规模超4亿人，拥有超大规模市场和丰富的技术应用场景。民生日用的现实需求与日俱增，各行业对新技术、新产品翘首以盼。当供给侧"贴合"需求侧，当生产与消费深度互动，创新的"星星之火"就能以"燎原之势"形成新的产业蓝海。

往深层看，坚持科技向善惠民，不仅关乎民生改善，更关乎科技创新的未来。当今时代，人工智能、生物医药、基因编辑等新技术在激发发展新动能的同时，也对社会治理、法律规则等各方面带来新的冲击。比如，大语言模型处理、生成数据时，可能侵犯个人隐私；换脸、拟声等技术可能被用来诈骗、制造虚假信息。因此，科技创新已经不只是一个技术课题，而是具有了更鲜明的价值属性、治理属性、伦理属性。只有坚持科技向善惠民，并在这样的价值基准上完善社会治理、法律规则，才能在防范社会风险的同时让科技创新的正向效应最大化。

中国式现代化，民生为大。科技创新最终要落到"人"上。新征程上，抓住一切有利时机，利用一切有利条件，推动更多好技术变成新应用，定能以科技创新的累累硕果惠及广大群众，创造更加美好的生活。

摘编自《人民日报》(2025年10月15日)

### 材料2

前不久，到宇树科技参观，负责人特意介绍起脚下的一段楼梯：低处的台阶边缘多处破损，越往上，破损痕迹越少。原来，这是训练机器人时留下的。一开始，机器人可能走不了几步就会摔倒；经过不断调试改良，越走越顺利，爬楼已经不在话下。这段意义独特的楼梯，镌刻成长之路，照见创新之路。

创新是一个不断试错的过程，望而却步不行，盲目蛮干也不行。在我国航天领域，有一套独特的工作方法——归零。哪方面出现问题时，就从技术和管理两个层面，对问题进行定位、分析、复现、纠正、举一反三，确保从根本上消除问题。正因为不怕试错、善于纠错，才成就了探月工程六战六捷的佳绩。有位科研人员说得好："做科研最不怕的就是'问题'，有'问题'的地方正是科研可挖的'井'。"大胆假设、认真求证，在总结经验教训中再出发，才能通往成功彼岸。

不断试错的创新，需要更多包容与支持。在2025年世界人形机器人运动会上，一些机器人"选手"摔倒等情况引发讨论。事实上，创新就有可能跌跌撞撞。新兴产业与未来产业在孕育早期，也很难做到"零失误""零瑕疵"。从稚嫩到成熟，从踉跄到奔跑，这是事物发展的必经阶段。过去，新能源汽车续航短是个困扰，但今天"里程焦虑"正在技术的升级下不断得到缓解，新能源汽车越来越受欢迎。近来，多地出台科技创新"容错"新规，更多耐心资本充当企业"长跑搭档"。营造好鼓励创新、宽容失败的环境，给予创新创业"十年不鸣"的底气，才能浇灌出"一鸣惊人"的硕果。

摘编自《人民日报》(2025年10月9日)

## 问题

### (1) 如何理解"向善惠民，始终是科技创新的价值取向"？

#### 原理一：真理尺度与价值尺度的统一

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 任何成功的实践都是真理尺度和价值尺度的统一。 |
| 第二步：解释考点 | 实践的真理尺度是指实践中人们必须遵循真理的指导。实践的价值尺度是指在实践中人们按照自己的尺度和需要去认识世界和改造世界。 |
| 第三步：阐明关系 | 实践的真理尺度和价值尺度是辩证统一的关系。一方面，价值尺度必须以真理为前提。另一方面，人类自身需要的内在尺度，推动着人们不断发现新的真理。 |
| 第四步：方法论 | 无 |
| 第五步：结合材料 | 只有推动科技创新向善惠民，才能符合价值尺度，从而达到真理尺度和价值尺度的统一。 |

#### 原理二：科学技术的双刃剑属性

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 科学技术是推动社会文明进步的重要力量。但是，科学技术的发展是一把"双刃剑"。 |
| 第二步：解释考点 | 科学技术在促进经济和社会发展造福于人类的同时，也可能在一定条件下对人类的生存和发展带来消极后果。科学技术的作用会受到主客观条件的制约。 |
| 第三步：阐明关系 | 无 |
| 第四步：方法论 | 我们必须要用合理的社会制度保障科学技术的正确运用，使科学技术为人类造福。 |

### (2) 为什么在鼓励创新的同时也要宽容失败？

#### 原理一：真理与谬误的辩证关系

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 真理与谬误、成功与失败是对立统一的。 |
| 第二步：解释考点 | 真理是人们对客观事物及其规律的正确反映；谬误是人们对客观事物及其规律的歪曲反映。 |
| 第三步：阐明关系 | 在确定的对象和范围内，真理和谬误是对立的，但是在一定条件下，它们又可以相互转化。真理在一定条件下可以转化为谬误，谬误在一定条件下也可以转化为真理。在批判谬误中发展真理，是谬误向真理转化的一种形式。 |
| 第四步：方法论 | 无 |

#### 原理二：否定之否定规律

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 否定之否定规律揭示了事物发展的前进性与曲折性的统一。 |
| 第二步：解释考点 | 发展的前进性体现在每一次否定都会把事物发展推向一个新阶段；发展的曲折性体现在事物发展过程中有暂时的停顿甚至倒退。这表明，事物的发展不是直线上升的，而是一个螺旋式上升、波浪式前进的过程。 |
| 第三步：阐明关系 | 无 |
| 第四步：方法论 | 无 |
