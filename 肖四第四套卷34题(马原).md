# 肖四第四套卷 34题(马原)

## 题目

**34. 结合材料回答问题：**

### 材料1

我们要学习陈云同志的科学思想方法和工作方法。坚持实事求是，是陈云同志的鲜明特点。他指出："要把我们的党和国家领导好，最要紧的，是要使领导干部的思想方法搞对头。"他提炼出"不唯上、不唯书、只唯实，交换、比较、反复"这一充满唯物辩证法的"十五字诀"。他善于从战略上思考问题，强调："应该站得高一点，看得远一点，要上屋顶，不要老呆在地下室。""要拿出一定时间'踱方步'，考虑战略性的问题。"他把调查研究作为决策的重要前提，强调："领导机关制定政策，要用百分之九十以上的时间作调查研究工作。"

摘自习近平《在纪念陈云同志诞辰120周年座谈会上的讲话》

### 材料2

正值炎热署期，吉林长白山的山地旅游成功"破圈"，与冰雪季拥有同样的热度。

再细细观察，今天很多滑雪场转变思路，摆脱了季节"魔咒"。同一片山坡，冬日可以滑雪，夏日则能观景；同一架缆车，雪季运送滑雪者，暖季变身观光线。当发展的视野不再局限于单一冰雪资源，而是顺应消费需求，创新供给，那么资源"闲置期"就转化为了发展"机遇期"。

从"一季养四季"到"四季皆旺季"，正是以认知升维实现资源再利用、打开发展新思路的一个生动体现。换一个看待事物的方式，就可能会发现资源排列组合的新形式、生产要素之间的新配置，从而产生新的发展思路、找到新的发展路径。

习近平总书记在浙江工作时科学阐释"地瓜经济"理论，深刻指出"必须跳出浙江发展浙江，在大力引进各种要素的同时打到省外去、国外去，利用外部资源、外部市场实现更大的发展"。以认知升维打开发展思路，正是一个跳出自身、着眼全局、从更大视野谋划发展思路的过程。比如，风光电让西北戈壁滩成了优质发电场所，数字经济发展让西南山区变身数据中心，战略思维、大局意识让我们可以重新认识自己的长板与短板，进而实现由短变长的蝶变。

## 问题

### (1) 分析"不唯上、不唯书、只唯实"这一论断所蕴含的哲学原理

#### 原理一：世界物质统一性

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 世界的物质统一性原理是辩证唯物主义最基本、最核心的观点，是马克思主义的基石。 |
| 第二步：解释考点 | 世界是统一的，世界的本原只有一个，世界统一于物质。 |
| 第三步：阐明关系 | 无 |
| 第四步：方法论 | 该原理要求我们想问题、办事情必须做到一切从实际出发。一切从实际出发，从根本上说，就是要从客观事物存在和发展的规律出发，在实践中按照客观规律办事。 |
| 第五步：结合材料 | "不唯上、不唯书、只唯实"就是反对盲从权威，反对本本主义，一切从实际出发。 |

#### 原理二：实践决定认识

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 实践决定认识，实践是认识的来源、动力、目的和检验认识真理性的唯一标准。 |
| 第二步：解释考点 | 实践是人们改造自然和社会的有目的、有意识的活动。实践决定认识，意味着认识的产生、认识的进行、认识的结果都取决于实践；实践是认识的目的，认识是为了指导实践；实践是检验真理的唯一标准，决定了真理的相对性和绝对性、有条件性和无条件性。 |
| 第三步：阐明关系 | 无 |
| 第四步：方法论 | 无 |

#### 原理三：真理的绝对性与相对性

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 任何真理都是绝对性与相对性的统一。 |
| 第二步：解释考点 | 真理的绝对性是指真理主客观统一的确定性与发展的无限性。真理的相对性是指人们在一定条件下对客观事物及其规律的正确认识总是有限度的，不完善的。 |
| 第三步：阐明关系 | 真理的绝对性与相对性是辩证统一的。一方面，二者相互包含，绝对之中有相对，相对之中有绝对；另一方面，任何真理都处于真理的相对性走向绝对性转化的过程中。 |
| 第四步：方法论 | 承认真理的绝对性与相对性的辩证统一，就必须以科学态度对待一切真理，既要反对绝对主义，又要反对相对主义。绝对主义在实际工作中就表现为教条主义和思想僵化。 |

### (2) "地瓜经济"理论启示我们"跳出自身、着眼全局、从更大视野谋划发展思路""突破思维局限，从更高的维度、更长的周期来看自身、看当下"，这体现了怎样的思维能力？

#### 辩证思维能力

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 辩证思维是在分析和解决问题时贯彻运用唯物辩证法的基本观点和方法。 |
| 第二步：解释考点 | 提高辩证思维能力，要求我们联系地而不是孤立地、发展地而不是静止地、全面地而不是片面地、系统地而不是零散地观察事物。 |
| 第三步：阐明关系 | 无 |
| 第四步：方法论 | 无 |

#### 系统思维能力

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 系统思维能力是全面系统地分析和处理问题的能力。 |
| 第二步：解释考点 | 坚持系统观念，提高系统思维能力，要把握好全局与局部、当前与长远的关系，加强前瞻性思考、全局性谋划、战略性布局和整体性推进。 |
| 第三步：阐明关系 | 无 |
| 第四步：方法论 | 无 |

#### 战略思维能力

| 步骤 | 内容 |
|------|------|
| 第一步：抛出考点 | 战略思维能力强调思维的整体性、全局性、长期性，是高瞻远瞩、统揽全局、善于把握事物发展总体趋势和方向的能力。 |
| 第二步：解释考点 | 培养和提高战略思维能力，要求我们不断开阔视野，学会站在战略和全局的高度观察和处理问题。 |
| 第三步：阐明关系 | 无 |
| 第四步：方法论 | 无 |
| 第五步：结合材料 | "地瓜经济"理论通过跳出自身、着眼全局、从更大视野谋划发展思路，体现了辩证思维、系统思维和战略思维的统一。从"一季养四季"到"四季皆旺季"，从西北戈壁滩到发电场，从西南山区到数据中心，充分体现了以更高的维度、更长的周期看问题的战略思维能力。 |
