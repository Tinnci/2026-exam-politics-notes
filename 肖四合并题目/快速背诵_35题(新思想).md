## 📌 四套卷分析（核心金句+逻辑）

### 第一套卷：高质量发展与创新

**题干关键词**：杭州"六小龙"、新质生产力、比学赶超、错位发展

**必背核心金句**：
1. "新质生产力本质是**先进生产力**，是符合新发展理念的"
2. "**不争先就难领先**"
3. "要学习借鉴，但**不是简单复制**"
4. "要**立足自身资源禀赋，找准定位**"
5. "一体化不是同质化，须**错位发展、发挥比较优势**"

**答题逻辑框架**（背核心词+句式）：
- ✅ **新质生产力的地位**：是先进生产力，体现新发展理念
- ✅ **地区竞争的方式**：不是复制，而是根据自身特色发展长板
- ✅ **区域协调的思路**：错位发展、比较优势、做大长板而非补短板

**可现场发挥的部分**：
- 关于杭州的具体案例、苏州生物医药、南京特种钢的细节 → 抄材料即可

---

### 第二套卷：竞争与合作（共赢思维）

**题干关键词**：两家餐厅互相分享客源、"三赢"、内卷、共赢

**必背核心金句**：
1. "**市场欢迎竞争**，但竞争不能被窄化为'你取代我、我取代你'"
2. "当人们聚焦'**能一起得到什么**'而非'你会抢走我什么'时，竞争的**攻防模式切换到探索模式**"
3. "要**打破狭隘的思维局限**，跳出自家的'一亩三分地'"
4. "**中国市场足够大，容得下亿万经营主体共生共赢**；世界经济足够宽广，可任中外品牌千帆竞秀、百舸争流"

**答题逻辑框架**：
- ✅ **问题诊断**：内卷源于狭隘思维（零和博弈）
- ✅ **解决方案**：从对抗竞争→合作共赢（思维转变）
- ✅ **可能性**：市场足够大，资源足够多

**考点对应**：
- **矛盾的同一性与斗争性**：竞争是斗争性，合作是同一性，二者辩证统一
- **系统优化方法**：从局部视角→整体视角，优化资源配置

### 第三套卷：创新驱动与比较优势

**题干关键词**：地瓜经济、"跳出浙江发展浙江"、认知升维、西北戈壁→光伏、山区→数据中心

**必背核心金句**：
1. "**必须跳出浙江发展浙江**，在大力引进各种要素的同时**打到省外去、国外去**"
2. "以**认知升维**打开发展思路，是**跳出自身、着眼全局、从更大视野谋划**的过程"
3. "**换一个看待事物的方式**，就可能发现资源排列组合的新形式、生产要素的新配置"
4. "**战略思维、大局意识让我们重新认识自己的长板与短板，实现由短变长的蝶变**"

**答题逻辑框架**：
- ✅ **视野升维**：从单一资源→多维利用，从区域→国际视野
- ✅ **思维转变**：战略思维、大局观、系统观
- ✅ **结果**：短板变长板（西北戈壁变电场、山区变数据中心）

**考点对应**：
- **战略思维、系统思维、创新思维**：联系地、全面地、创新地看问题
- **新发展理念**：创新（突破常规）、开放（走出去）、共享（要素流动）

### 第四套卷：高质量发展的确定性

**题干关键词**："以高质量发展的确定性应对各种不确定性"、发展战略、宏观政策、市场空间、发展韧性、投资环境

**必背核心金句**：
1. "**新质生产力本质是先进生产力**"
2. "**高质量发展是能够很好满足人民日益增长美好生活需要的发展**"
3. "**实现高水平科技自立自强是国家强盛和民族复兴的战略基石**"
4. "**党的领导直接关系中国式现代化的根本方向、前途命运、最终成败**"

**答题逻辑框架**（采用题目给出的五个方面）：
- ✅ **发展战略的确定性** → 中国特色社会主义方向不动摇
- ✅ **宏观政策的确定性** → 以人民为中心、新发展理念
- ✅ **市场空间的确定性** → 14亿人口、中等收入群体超4亿
- ✅ **发展韧性的确定性** → 制度优势、党的领导
- ✅ **投资环境的确定性** → 改革开放、法治化营商环境

## 🟡 第二级：背熟逻辑关键词（现场转换）

### 新思想答题的"万能句式"

| 句式模板 | 填空示例 | 适用题型 |
|---------|---------|---------|
| "A是B的____，是C的____" | 党的领导是中国特色社会主义**最本质的特征**，是制度**最大优势** | 地位类 |
| "A为B提供了____支撑" | 新质生产力为高质量发展提供**战略支撑** | 作用类 |
| "只有A，才能B；B是A的____" | 只有贯彻新发展理念，才能推动高质量发展；新发展理念是高质量发展的**指导原则** | 必要性 |
| "A既是____，也是____；是____的重要内容" | 高质量发展既是**经济问题**，也是**政治问题**；是推进中国式现代化的**首要任务** | 重要意义 |
| "要把A和B____起来" | 要把**发展和安全****统一起来** | 协调类 |

### 高频核心词汇（必须掌握）

**"最"字句关键词**（政治定性，不能改）：
- 最本质特征 / 最大优势 / 最伟大的革命 / 最深层的利益
- 最重要的 / 最根本的 / 最关键的

**"必由之路"类**（方向性金句）：
- 开放是____的必由之路
- 改革是____的必由之路
- 创新是____的必由之路

**"根本保证/根本动力"类**：
- A是B的**根本保证** / **根本出发点** / **战略基石**
- A为B提供**精神支撑** / **物质基础** / **制度保障**
